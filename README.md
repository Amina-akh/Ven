# «Венера-1» — сайт-презентация

Интерактивная презентация проекта (12 полноэкранных слайдов, 3D-фон, плавные движения камеры и аппарата).

## Запуск

```bash
npm install
npm run dev
```

Сборка: `npm run build`, предпросмотр: `npm run preview`.

## Материалы для команды

Тексты для **пяти** выступающих и таблица «кто на каком слайде» — в папке [`prezentacija-5-ucastnikov/`](prezentacija-5-ucastnikov/README.md).
